## 1. Información que Recopilo

### 1.1 Información que Proporcionas Directamente

#### 1.1.1 Datos de Discord

Cuando usas Hoshiko, recopilo:

- Tu ID único de usuario de Discord
- Tu nombre de usuario y discriminator
- Tu avatar y foto de perfil
- Tu ID de servidor (guild) donde usas Hoshiko
- Tu rol y posición en el servidor

#### 1.1.2 Contenido que Creas

- Mensajes publicados en confesiones
- Configuración de tu perfil
- Datos de rachas (grupos que creas o juntas)
- Configuración AFK y mensajes personalizados
- Sugerencias e ideas que envías

#### 1.1.3 Interacciones

- Comandos que ejecutas
- Interacciones con otros usuarios (abrazos, besos, etc.)
- Respuestas a encuestas y votaciones
- Configuración de preferencias

### 1.2 Información Recopilada Automáticamente

#### 1.2.1 Datos de Uso

- Historial de comandos usados
- Frecuencia de uso de funcionalidades
- Patrones de actividad
- Tiempos y fechas de uso

#### 1.2.2 Datos Técnicos

- Direcciones IP (cuando se conectan a mi servidor)
- Tipo de navegador y dispositivo (si aplica)
- Errores y logs técnicos
- Metadatos de mensajes

#### 1.2.3 Datos de Moderación

- Advertencias recibidas
- Acciones disciplinarias
- Registros de mensajes eliminados
- Historial de muteos y baneos

### 1.3 Información de Servidores

#### 1.3.1 Configuraciones

- Configuración de canales de confesiones
- Configuración de moderación
- Configuración de IA
- Roles y permisos vinculados a Hoshiko

#### 1.3.2 Datos de Rachas

- Grupos de rachas
- Miembros y su actividad
- Estadísticas de rachas
- Días consecutivos

### 1.4 Información de la IA

#### 1.4.1 Conversaciones

- Mensajes enviados a la IA
- Respuestas generadas
- Contexto de conversación

#### 1.4.2 Memoria de Usuario

- Datos que la IA aprende sobre ti
- Preferencias inferidas
- Historial de interacciones

## 3. Almacenamiento de Datos

### 3.1 Ubicación

Los datos se almacenan en:

- **Base de datos:** MongoDB Atlas (servidores en Estados Unidos y Europa)
- **Logs:** Servidores de Hoshiko
- **Cache:** Memoria del servidor donde corre Hoshiko

### 3.2 Medidas de Protección

Para proteger tu información:

- Contraseñas de base de datos en variables de entorno
- Conexiones encriptadas (TLS/SSL)
- Acceso restringido a la base de datos
- Backups regulares

### 3.3 Limitaciones

- Como soy una sola persona, los recursos para seguridad son limitados
- Utilizo servicios de terceros confiables (MongoDB Atlas)
- No puedo garantizar el mismo nivel de seguridad que grandes corporaciones

## 4. Compartir Información

### 4.1 Principios Generales

- **NO vendo** tu información personal a nadie
- **NO alquilo** tu información personal
- **NO comparto** tu información con fines de marketing

### 4.2 Compartición con Terceros

#### 4.2.1 Proveedores de Servicios

Comparto datos con:

- **MongoDB Atlas:** Almacenamiento de base de datos
- **Google (Gemini):** Procesamiento de IA (tus mensajes pueden ser enviados a su API)
- **Discord:** Plataforma subyacente (su política de privacidad aplica)

#### 4.2.2 Administradores de Servidores

Los propietarios y administradores de servidores pueden ver:

- Configuraciones del servidor
- Logs de moderación de ese servidor
- Estadísticas del servidor

#### 4.2.3 Confesiones

En el sistema de confesiones:

- Tu identidad **NO se muestra** a otros usuarios
- **SÍ se guarda internamente** para moderación
- Los admins del servidor **pueden verla** si es necesario

### 4.3 Divulgación Legal

Puedo divulgar información si:

- La ley lo requiere
- Es necesario para cumplir con procedimientos legales
- Es necesario para proteger mis derechos
- Es necesario para prevenir actividad ilegal

## 5. Retención de Datos

### 5.1 Períodos de Retención

| Tipo de Dato               | Período de Retención                    |
| -------------------------- | --------------------------------------- |
| Datos de servidor (config) | Mientras el bot esté en el servidor     |
| Historial de comandos      | 30 días                                 |
| Logs de moderación         | 90 días                                 |
| Confesiones                | Hasta eliminación manual                |
| Datos de rachas            | Indefinida (necesarios para el sistema) |
| Advertencias               | Indefinida (necesarias para moderación) |
| Datos de IA/Memoria        | Hasta eliminación por solicitud         |

### 5.2 Eliminación Automática

Algunos datos se eliminan automáticamente:

- Logs de comandos después de 30 días
- Datos de interacción después de 30 días

### 5.3 Eliminación Manual

Puedes solicitar eliminación de tus datos personales en cualquier momento.

### 5.4 Conservación Legal

Puedo retener datos por más tiempo si:

- Hay obligaciones legales de conservación
- Hay una disputa legal en curso
- Es necesario para ejercer derechos legales

## 9. Privacidad de Menores

### 9.1 Restricción de Edad

Hoshiko **no está diseñado para usuarios menores de 13 años** (o 16 años en la Unión Europea según las leyes locales).

### 9.2 Recopilación de Datos de Menores

Si determino que he recopilado información de un usuario menor de 13 años:

- Eliminaré dicha información inmediatamente
- No utilizaré dicha información para ningún propósito
- No proporcionaré ningún servicio a dicho usuario

### 9.3 Responsabilidad de los Administradores

Los propietarios y administradores de servidores son responsables de:

- Garantizar que usuarios menores de edad tengan permiso parental
- Configurar filtros apropiados para menores
- Moderar contenido apropiado para menores

### 9.4 Padres y Tutores

Si eres padre o tutor de un menor que usa Hoshiko:

- Habla con el menor sobre seguridad en línea
- Supervisa su uso de Discord y bots
- Contacta a v.sxn si tienes preocupaciones
